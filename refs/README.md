# Visual References

Design direction: **dark-grey minimal**. Not pure black — balance of dark greys with white accents. Clean, functional, no clutter.

---

## ref-01-dark-grey-minimal.png

Dark UI with layered grey surfaces and white text. Key takeaways:

**Palette extracted:**

| Role | Hex | Usage |
|---|---|---|
| Deepest bg | `#0f0f0f` | Wallpaper area, outermost layer |
| Base bg | `#141414` | App window background |
| Surface | `#1c1c1c` | Sidebars, panels |
| Overlay / card | `#252525` | Cards, elevated elements |
| Hover | `#2e2e2e` | Hover states |
| Border | `#333333` | Borders, dividers |
| Text primary | `#ffffff` | Headings, labels |
| Text secondary | `#909090` | Muted text, hints |

**Principles:**
- No pure `#000000` backgrounds — minimum is `#0f0f0f`
- Grey layering creates depth without colour
- Borders are visible but subtle (`#333`)
- White is used as accent, not background
- Folder/file icons: dark grey (not colourful)
- High contrast text on dark surface = readable without being harsh
